# 好不好玩 · 验收宪法（solo）

> 2026-07-23 与用户对齐：**旧核心「拖云搬水」已否决（会玩但不想再来）。**  
> 方向 **B = 1+4**：天气玩具 + 山谷守护。  
> 验收人：**仅用户本人**（不要求儿童样本）。

## 北极星（唯一硬门槛）

**玩约 3 分钟后：你还想不想再来一把？**

- **过**：想，或已经下意识点了「再来」  
- **不过**：没意思 / 不想再点 → **整段设计作废**，不堆关卡、不堆 meta

次要（过北极星后再看）：

- 会不会**主动再开一局**（同一会话）  
- 明天还想不想打开  

## 什么叫「有意思」（C 条，新核心至少 3/4）

| # | 标准 | V2 原型如何体现 |
|---|------|----------------|
| C1 | 即时后果 | 雨落处湿、不雨处晒、枯裂纹立刻出现 |
| C2 | 权衡 | 同时多块田变干，瞄哪块 / 下多大 |
| C3 | 变化 | 正午蒸发更快，同样下雨效果不同 |
| C4 | 可读推进 | 顶部 🌿 存活数、白天进度条、黄昏胜负 |

## 明确不算成功

- 测试绿、关卡多、系统多  
- 「有教育意义」但你自己不想再玩  
- 文档写得很漂亮  

## 技术红线（保留）

零外部资源 · 可静态部署 · 最简路径 6 岁可摸 · 无 IAA 体力广告  

## 旧版战役

**冻结**：选关仍可玩 v1，但不再作为「是否好玩」的评价对象。  
评价入口：**选关页 → ✨ 试玩新核心**。

## 你怎么评这个原型

1. `npm run dev` → 建角色/选档 → 选关页点 **✨ 试玩新核心**  
2. 拖动画布瞄准，按住右下角 ☔ 下雨  
3. 撑到黄昏 or 3 块田旱死  
4. 只回答一句：**还想再来一把吗？**（想 / 不想 / 为什么）
